## 🎮 Interactive Demo

Try CyberCoder right now:

```bash
# Install
npm install -g cybercoder-cli

# Start
cybercoder

# Login (required like Claude Code)
/login

# Add your API key
/secret set ANTHROPIC_API_KEY your-key

# Start coding!
"Help me build a React todo app"
```
