HOW TO VERIFY / DEBUG
═══════════════════════════════════════════

  pnpm -r --parallel run typecheck   # all 9 packages must pass
  pnpm --filter @cybermind/cli run build
  node packages/cli/bin/cybermind.mjs --version

If typecheck fails: most issues are stale IDE state. Run the command above
to see what tsc actually sees. The IDE may show ghost "Cannot find module" lints
that disappear once you re-open the workspace.

Logs go to ~/.cybermind/logs/cybermind-YYYY-MM-DD.log (set CYBERMIND_LOG_LEVEL=debug
for verbose output). Trust store is at ~/.cybermind/trust.json. Encrypted secrets
vault is at ~/.cybermind/secrets.enc.
